### Academic Grade Management System

A Java-based academic grade management program that processes student scores and generates semester statistics.

**Concepts applied:**

- One-dimensional arrays

- Looping and conditional statements

- Menu-driven program

- Data searching

- Grade calculation

- Basic statistical processing

**Features:**

- Input and validate student grades

- Display all course results

- Search courses by name

- Find highest and lowest scores

- Calculate semester GPA

- Display grade distribution
